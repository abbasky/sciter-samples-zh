sciter.node - basic NodeJs Sciter integration

Prerequisites:

  • /sdk/bin/64/sciter.node - Native Sciter Module for NodeJs, exposes Sciter namespace object (in terms of Node). 
  • /sdk/demos/sciter.node - source code of sciter.node DLL.

Sample files:

  • helloworld.js - basic demo:
    • creates Sciter.View (window);
    • loads HTML from string;
    • runs message pump loop;
  • ui-methods.js - calling Sciter methods from NodeJs (JavaScript) side
    • creates Sciter.View and loads /ui/ui-methods.htm file in it.
    • calls function defined in ui-methods.htm from NodeJs side.
    • runs message pump loop;
  • node-methods.js - calling NodeJs (JavaScript) methods from Sciter side
    • creates Sciter.View and loads /ui/node-methods.htm file in it.
    • call JavaScript function defined in NodeJs from Sciter side.
  • sciter.js - loads native sciter.node module
    • returns Sciter namespace;
    • adds Sciter.run() message pump "loop" to native Sciter namespace. Sciter.run() call is required for UI operation.

Native sciter.node module

Exports:

  • View class - Sciter window;
  • pumpMessages() function - native function, executes PeekMessage/DispatchMessage;

Sciter.View class - represents Sciter window on JavaScript/Node side. It allows to call SciterScript functions from JavaScript (Node side) as to call JavaScript functions (a.k.a. backend) from SciterScript side (a.k.a. frontend).

Methods:

  • {constructor} - new Sciter.View(); - constructs instance of sciter window, window is in hidden state initially;
  • loadHtml(html:string) - loads HTML from the string into the window;
  • loadFile(path:string) - loads HTML from file;
  • expand([fullScreen:true | false]) - shows the window;
  • collapse() - hides the window;
  • call(funcName:string[, args...]): value - calls sciter script function defined in document loaded into the window. JavaScript -> SciterScript call;

JavaScript instance of Sciter.View class may have arbitrary JavaScript methods added to it. These methods are callable from SciteScript side as view.method(args) .
